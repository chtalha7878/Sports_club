# UAE Lions Sports Club

A modern React website for UAE Lions Sports Club, built with Vite and Tailwind CSS.

## Sections

- Home
- About
- Sports
- Events
- Contact

## Tech Stack

- React
- Vite
- Tailwind CSS
- Lucide React

## Local Development

```bash
pnpm install
pnpm run dev
```

## Build

```bash
pnpm run build
```
